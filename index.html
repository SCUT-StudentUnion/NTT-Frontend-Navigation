    # 前言

    这是一篇简单的、有集合性质的属于 NTT 前端的教程

    虽然说自己造轮子好像没有什么必要，而且以我们的水平肯定不会写得比大佬们好，但是其实很多人在学习的过程中都会被这样那样的教程搞得有点选择困难，那我们就来写一个属于 NTT 的前端导航吧

    也就是说，我们是以目前 NTT 的前端技术栈出发，介绍如何搭建目前的技术栈

    另外，本向导还打算额外解决大家在从零入门的时候，我自己观察到的一些很有趣的问题，这些我们放在前面的部分讲

    那我们就开始吧
